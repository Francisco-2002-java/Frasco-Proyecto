❤️ Nuestro Rincón Especial

Este es un proyecto personal creado con mucho cariño. Se trata de un Frasco Virtual de Motivos donde guardo todas las razones por las que eres especial para mí.

✨ Características

Frasco Interactivo: Haz clic en el frasco o en los corazones para descubrir un mensaje.

Música Personalizada: El sitio reproduce nuestra playlist especial de fondo (puedes gestionarla desde el icono de la nota musical).

Mensajes en Tiempo Real: Los motivos y la música se guardan en la nube, por lo que siempre estarán actualizados.

Diseño Adaptable: Se ve perfecto tanto en móvil como en ordenador.

🚀 Cómo usar

Abre el enlace de GitHub Pages.

Interactúa con el frasco para leer los mensajes.

Usa el formulario de la parte inferior para añadir nuevos motivos cuando quieras.

Usa el botón flotante de la derecha (icono musical) para añadir enlaces de canciones (.mp3 o .mp4).
